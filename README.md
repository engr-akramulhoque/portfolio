## How to run the script

```bash
    git clone https://github.com/engr-akramulhoque/portfolio.git
```

```bash
    cd portfolio
```

```bash
    npm install --legacy-peer-deps
```

```bash
    npm run dev
```

### Deploy the portfolio on github pages

```bash
    npm run deploy
```
